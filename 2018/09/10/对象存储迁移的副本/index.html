青云控制台
  1. 创建桶
青云服务器（安装qscamel）
  1. 下载 wget https://github.com/yunify/qscamel/releases/download/v2.0.5/qscamel_v2.0.5_linux_amd64.tar.gz
  2. 解压，重命名为qscamel，可直接使用。
  3. 编辑yaml配置文件
  4. 停止aws和mysql，防止文件被进程占用，防止产生新文件。
  5. 备份
  6. 扩容，目前剩余空间不够打包空间。(可以挂载新硬盘。)
  7. 打包，零散文件传输太慢
  8. 跨区传输，加大带宽到200M (5M/5.6Gb/3h 200M/5.6Gb/4m)
  9. qscamel run
华为云控制台
  1. 创建桶
  2. 创建迁移任务，将青云的桶数据迁移到华为云（5M/5.6Gb/1.5h）
华为云服务器（安装obscmd）
  1. 安装openssl，https://www.cnblogs.com/cuijianxin/p/6408757.html
  2. 安装openssl-devel，yum -y install openssl-devel
  3. 升级python2.6.6到2.7.14
  4. 下载obscmd，wget http://static.huaweicloud.com/upload/files/obs/cmd.zip
  5. unzip cmd.zip
  6. 参考配置文档修改config.dat，然后下载：python run.py(5.6Gb/2m)
  7. 计算摘要，确认文件完全一致
